# Police Video Notes

英語の海外警察関連YouTube動画を日本語で理解するためのローカルノートアプリです。

## 使い方

1. `index.html` をブラウザで開く、またはローカルサーバーで開きます。
2. YouTube URL、タイトル、英語字幕や説明文を貼ります。
3. `日本語メモを作る` を押すと、要約枠、見どころ、用語メモ、AIに渡す翻訳依頼文が更新されます。
4. `依頼文コピー` を押して、ChatGPTやCodexに貼ると日本語翻訳を依頼できます。
5. `この動画を保存` でブラウザ内に動画ノートを保存できます。
6. `保存データをコピー` で、スマホのメモアプリなどにバックアップできます。

## スマホで使う

スマホのブラウザでアプリURLを開き、共有メニューからホーム画面に追加するとアプリのように起動できます。スマホ幅では、下の `入力` / `メモ` / `保存` タブで画面を切り替えます。

保存データはブラウザ内に保存されます。機種変更やブラウザ変更に備える場合は、`保存` タブの `保存データをコピー` を押して、コピーした文字をメモアプリなどに残してください。戻すときは、その文字を復元欄に貼って `貼ったデータで復元` を押します。

## ローカルサーバー

```bash
python3 -m http.server 8765
```

その後、ブラウザで `http://127.0.0.1:8765/` を開いてください。

## 現時点の範囲

この版は、YouTubeから字幕を自動取得したり、AI翻訳APIへ直接送信したりはしません。まずは字幕テキストを貼って、翻訳依頼文と動画ノートを整理するためのプロトタイプです。
